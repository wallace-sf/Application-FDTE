## Instalação
Primeiro vamos fazer o clone do projeto. Digite:
```sh
$ git clone https://github.com/wallace-sf/Application-FDTE.git
```
Instale as dependências:
```sh
# npm
$ npm i

# yarn
$ yarn
```

## Scripts
```sh
# Inicie o projeto em  de desenvolvimento
$ yarn start
$ npm start
```
